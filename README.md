# BLACKMODULE

BLACKMODULE est un prototype de filtrage des clients contre les listes de sanctions, PPE et listes internes.

## Fonctionnalités principales

- Import des listes de sanctions
- Matching client
- Génération des alertes
- Traitement des alertes
- Journalisation audit
- API externe sécurisée
- Scheduler de mise à jour des listes
