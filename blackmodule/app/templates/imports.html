AFB PPE · CSV

Importer la liste PPE AFB

Format CSV attendu :
nom, prenom, date_naissance, nationalite, pays, num_passeport, motif_sanction, date_inscription, statut

Exemple :
DOE,JOHN,1980-05-12,CAMEROUNAISE,CAMEROUN,P123456,Client PPE de test,2025-06-01,ACTIF
OFAC SDN · XML

Importer la liste OFAC SDN

Format attendu :
Fichier XML OFAC SDN contenant des balises comme :
sdnEntry, firstName, lastName, sdnType, akaList, dateOfBirthList, idList.

Les données seront normalisées dans sanction_entries avec la source OFAC_SDN.
ONU · XML

Importer la liste ONU

Format attendu :
Fichier XML ONU contenant des balises comme :
INDIVIDUAL, FIRST_NAME, SECOND_NAME, INDIVIDUAL_ALIAS, INDIVIDUAL_DATE_OF_BIRTH, INDIVIDUAL_DOCUMENT, ENTITY.

Les données seront normalisées dans sanction_entries avec la source ONU.
Union Européenne · CSV

Importer la liste Union Européenne

Format attendu :
nom, prenom, nom_complet, type_entite, date_naissance, nationalite, pays, num_passeport, motif_sanction, date_inscription, statut, alias

Les alias multiples doivent être séparés par un point-virgule : J DUPONT;JEAN D.
Union Européenne · XML

Importer la liste Union Européenne XML

Format attendu :
Fichier XML Union Européenne contenant des balises comme :
sanctionEntity, subjectType, nameAlias, wholeName, citizenship, birthdate, identification.

Les données seront normalisées dans sanction_entries avec la source UE.
OFSI UK · CSV

Importer la liste OFSI UK

Format attendu :
nom_complet, type_entite, date_naissance, nationalite, pays, num_passeport, adresse, motif_sanction, date_inscription, statut, alias

Les alias multiples doivent être séparés par un point-virgule : W BROWN;BILLY BROWN
OFSI / UKSL · CSV

Importer la UK Sanctions List

Format attendu :
Fichier CSV officiel UK Sanctions List contenant notamment :
Unique ID, OFSI Group ID, Name 1, Name 2, Name 3, Name 4, Name 5, Name 6, Date of Birth, Nationality, Passport Number, Regime Name, Sanctions Imposed

Les données seront normalisées dans sanction_entries avec la source UKSL.
OFAC Consolidated · XML

Importer la liste OFAC Consolidated

Format attendu :
Fichier XML OFAC Consolidated contenant des balises similaires à :
sdnEntry, firstName, lastName, sdnType, akaList, dateOfBirthList, idList.

Les données seront normalisées dans sanction_entries avec la source OFAC_CONSOLIDATED.
France Gel des Avoirs · JSON

Importer la liste France Gel des Avoirs

Format attendu :
nom, prenom, nom_complet, type_entite, date_naissance, nationalite, pays, num_passeport, decision, date_inscription, statut, alias

Les alias peuvent être une liste JSON : ["P DURAND", "PIERRE D."]
France Gel des Avoirs · XML

Importer la liste France Gel XML

Format attendu :
Fichier XML contenant les informations de gel des avoirs :
nom, prenom, alias, dateNaissance, nationalite, passeport, decision.

Les données seront normalisées dans sanction_entries avec la source FR_GEL.
