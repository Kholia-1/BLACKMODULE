Statut API

OPÉRATIONNELLE

L’API externe permet à une application tierce de lancer une vérification client contre les listes de sanctions actives de BLACKMODULE.

Toutes les requêtes doivent être authentifiées avec le header :

{{ api_key_label }}: {{ api_key_value }}

Sécurité

L’accès à l’API est protégé par une clé API. La clé réelle est stockée côté serveur dans le fichier .env et n’est jamais affichée en clair dans l’interface.

Chaque appel métier est journalisé dans le module d’audit afin d’assurer la traçabilité des consultations et des vérifications.
